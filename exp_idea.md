# Проект: Модель синхронизации осциллирующих нитей для адронов
## Текущее состояние (на 05.12.2024)

### Краткое описание
Мы разрабатываем феноменологическую модель адронов, где кварки представлены как **осциллирующие нити**. Ключевая гипотеза: **масса адрона определяется энергией синхронизации между колебаниями кварков**, а не просто суммой их масс.

### Основная формула
```
Для барионов: M = [Σ(m_q × f_q × a_q) + E_sync] × scale
Для мезонов:  M = [Σ(m_q × f_q × a_q) - E_sync] × scale
```
где:
- `m_q` - базовая масса кварка
- `f_q` - частота колебаний
- `a_q` - амплитуда колебаний  
- `E_sync` - энергия синхронизации
- `scale = 100` - масштабный коэффициент

### Ключевые физические инсайты (подтвержденные)
1. ✅ **Энергия синхронизации имеет РАЗНЫЙ знак** для барионов и мезонов
2. ✅ **coupling_neutron < coupling_proton** (~0.17-0.18 от протонного)
3. ✅ **Эффективные массы кварков в адронах**:
   - u-кварк: ~214 МэВ
   - d-кварк: ~351 МэВ
   - Отношение m_d/m_u ≈ 1.64
4. ✅ **Энергии связи**:
   - Протон: ~160 МэВ
   - Нейтрон: ~24-27 МэВ  
   - Пион π⁺: ~425 МэВ

### Функция энергии синхронизации
```python
def calculate_sync_energy(self):
    # Частотная когерентность
    freq_coherence = 1.0  # Упрощенно, все частоты близки
    
    # Фазовая когерентность
    phase_coherence_sum = 0
    for i in range(thread_count):
        for j in range(i+1, thread_count):
            diff = abs(phases[i] - phases[j]) % (2*π)
            diff = min(diff, 2*π - diff)
            
            if self.is_meson:
                phase_coherence_sum += cos(diff + π)  # Для мезонов
            else:
                phase_coherence_sum += cos(diff)      # Для барионов
    
    phase_coherence = (phase_coherence_sum / max_pairs + 1) / 2
    
    # Симметрия частицы
    symmetry = 1.0
    if self.particle_name == 'proton':
        symmetry = 1.1    # Два u-кварка → больше симметрии
    elif self.particle_name == 'neutron':
        symmetry = 0.95   # Меньше симметрии
    
    sync_energy = coupling * (0.6*freq_coherence + 0.4*phase_coherence) * symmetry
    return sync_energy
```

### Фазы колебаний для каждой частицы

- **Нейтрон (udd)**: фазы [0, π/2, π/2]  
- **Пион (u-anti-d)**: фазы [0, π]

```python
class ParticleModel:
    def __init__(self, composition, params):
        self.composition = composition  # ['u','u','d'] и т.д.
        self.params = params
        self.is_meson = len(composition) == 2
        
    def calculate_mass(self):
        base = self.calculate_base_mass()
        sync = self.calculate_sync_energy()
        
        if self.is_meson:
            return (base - sync) * 100
        else:
            return (base + sync) * 100
```
