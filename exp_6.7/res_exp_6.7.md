МОДЕЛЬ v6.7 - РАДИКАЛЬНОЕ РЕШЕНИЕ
Фиксированные легкие кварки + отдельные параметры для странных
================================================================================

ФУНДАМЕНТАЛЬНАЯ ПРОБЛЕМА v6.6:
  1. Невозможно одновременно описать легкие и странные частицы
  2. Для каонов нужны огромные энергии связи (>60 единиц)
  3. Текущая формула ограничивает coupling ~5-7

РЕШЕНИЕ v6.7:
  1. Фиксируем параметры u и d из v6.1 (идеально работают)
  2. Вводим отдельный масштаб для странных частиц
  3. Разрешаем coupling_meson_strange до 50
  4. Используем scale_strange для преобразования единиц

МАТЕМАТИЧЕСКАЯ СУТЬ:
  Для легких частиц: M = (база ± синхр) × 100
  Для странных частиц: M = (база ± синхр) × scale_strange
  где scale_strange ~ 10-20 (намного меньше 100)

ФИЗИЧЕСКИЙ СМЫСЛ:
  s-кварк в адронах имеет иную 'эффективную массу'
  Энергии связи для странных частиц могут быть другими
  Разные масштабы = разные единицы измерения для разных секторов

ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ:
  1. Протон, нейтрон, пион: ошибка < 0.1% (фиксировано)
  2. Каоны: ошибка < 3%
  3. Λ⁰: ошибка < 5%
  4. scale_strange ~ 10-15
  5. coupling_meson_strange ~ 15-30
