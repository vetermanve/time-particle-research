### Основная формула
```
Для барионов: M = [Σ(m_q × f_q × a_q) + E_sync] × scale
Для мезонов:  M = [Σ(m_q × f_q × a_q) - E_sync] × scale
```
где:
- `m_q` - базовая масса кварка
- `f_q` - частота колебаний
- `a_q` - амплитуда колебаний  
- `E_sync` - энергия синхронизации
- `scale = 100` - масштабный коэффициент

### Функция энергии синхронизации
```python
def calculate_sync_energy(self):
    # Частотная когерентность
    freq_coherence = 1.0  # Упрощенно, все частоты близки
    
    # Фазовая когерентность
    phase_coherence_sum = 0
    for i in range(thread_count):
        for j in range(i+1, thread_count):
            diff = abs(phases[i] - phases[j]) % (2*π)
            diff = min(diff, 2*π - diff)
            
            if self.is_meson:
                phase_coherence_sum += cos(diff + π)  # Для мезонов
            else:
                phase_coherence_sum += cos(diff)      # Для барионов
    
    phase_coherence = (phase_coherence_sum / max_pairs + 1) / 2
    
    # Симметрия частицы
    symmetry = 1.0
    if self.particle_name == 'proton':
        symmetry = 1.1    # Два u-кварка → больше симметрии
    elif self.particle_name == 'neutron':
        symmetry = 0.95   # Меньше симметрии
    
    sync_energy = coupling * (0.6*freq_coherence + 0.4*phase_coherence) * symmetry
    return sync_energy
```
